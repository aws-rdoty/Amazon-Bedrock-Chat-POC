
        
        Question: How do you calculate ABV in beer making?
        
        Answer:  Here are the steps to calculate the alcohol by volume (ABV) percentage in beer:

1. Measure the original gravity (OG) of the wort before fermentation using a hydrometer. The OG reading estimates the amount of sugars and solids dissolved in the wort.

2. Measure the final gravity (FG) of the beer after fermentation using a hydrometer. The FG reading estimates the amount of residual sugars and solids left after the yeast has consumed most of the fermentable sugars.

3. Plug the OG and FG values into the following formula:

ABV = (OG - FG) x 131.25

Where:

OG = Original gravity
FG = Final gravity 

131.25 is a constant factor that accounts for the conversion between gravity points and ABV.

4. For example, if the OG is 1.048 and the FG is 1.012, then:

ABV = (1.048 - 1.012) x 131.25 = 4.4%

So the ABV of the beer is 4.4%.

The difference between the OG and FG readings gives you the amount of sugars fermented. Multiplying this by 131.25 estimates the alcohol percentage based on the sugars converted to alcohol by the yeast.
        
        Question: create me a python function that solves fizzbuzz
        
        Answer:  Here is a Python function that solves the FizzBuzz problem:

```python
def fizzbuzz(n):
  for i in range(1, n+1):
    if i % 3 == 0 and i % 5 == 0:
      print("FizzBuzz")
    elif i % 3 == 0:
      print("Fizz")
    elif i % 5 == 0:
      print("Buzz")
    else:
      print(i)
```

The FizzBuzz problem typically goes:

For numbers 1 to n,
- if the number is divisible by 3, print "Fizz"
- if the number is divisible by 5, print "Buzz" 
- if the number is divisible by both 3 and 5, print "FizzBuzz"
- else, just print the number

So this function loops through the numbers 1 to n, checking each number's divisibility by 3 and 5 and printing the appropriate output.

To use it:

```python
fizzbuzz(15)
```

This will print out the FizzBuzz pattern for numbers 1 to 15.
